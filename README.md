Product Management System

A simple Product Management System built using **Node.js, Express, MySQL, HTML, CSS, and JavaScript**.

## Features
- User login
- Add, edit, delete products
- Product listing and summary
- Responsive UI

## Clone Project

```
git clone https://github.com/midhinababu/Product-Management-System.git
```

## Install and run server
```
cd Product-Management-System  
npm install  
node app.js
```

## Databse setup

The dump file is present in the root level of the project folder and can be used using this command

```
mysql -u root -p < product_db.sql

```

## Accessing the application

Open the browser and navigate to this url

```
http://localhost:3000/login.html
```

For simplicity the username is kept as *testuser* and the password is kept as *test1234*
